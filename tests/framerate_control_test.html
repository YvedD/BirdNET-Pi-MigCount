How it works:

The slider controls the frame-rate of the spectrogram:

  • Lower values (10-50ms): Faster refresh rate (smoother animation, higher CPU usage)
  • Medium values (100-150ms): Balanced refresh rate (good for most systems)
  • Higher values (200-500ms): Slower refresh rate (less smooth, lower CPU usage)
